## Как да смените placeholder-ите

В `index.html` потърсете коментарите `<!-- ЗАМЕНИ С: <img ... -->` и
заменете блока `<div class="image-placeholder ...">` с `<img>` таг.

Пример:
```html
<!-- Преди -->
<div class="image-placeholder">
  <i class="fa-solid fa-leaf"></i>
</div>

<!-- След -->
<img src="images/about.jpg" alt="За нас" />
```
